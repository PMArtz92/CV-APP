## Setup Development Environment in Local Machine
###1 Angular Client (Frontend)
To begin run the following commands in your terminal:

Install dependencies:

    $ cd client
    $ npm install

Install gem 'sass'

    $ gem install sass

Run development web-server:

    $ gulp serve
